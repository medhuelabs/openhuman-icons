# Guide

OpenHuman Icons is an open-source SVG icon system for anatomy, clinical, and health applications.

## Design Principles

- Start from a `24 × 24` canvas.
- Use a `2px` stroke with round caps and joins.
- Keep icons recognizable at their default size.
- Prefer simple medical metaphors over unnecessary detail.
- Use `currentColor` so icons inherit the surrounding interface color.
