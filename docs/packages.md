# Packages

OpenHuman Icons ships two focused packages in `v0.1`.

| Package                             | Purpose                                                                        |
| ----------------------------------- | ------------------------------------------------------------------------------ |
| `@medhuelabs/openhuman-icons`       | Framework-agnostic icon nodes, SVG serialization, metadata, and raw SVG assets |
| `@medhuelabs/openhuman-icons-react` | Tree-shakeable React components for every icon                                 |

Both packages are typed, tree-shakeable, and released under the ISC License.
